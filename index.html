Projects

Postgres Knowledge Base using Dify

This project is a custom Dify Plugin that enables intelligent Q&A over a code repository using PostgreSQL, pgvector, and LangChain. It powers a chatbot that answers technical questions based on embedded code files.

AdFusion:Personalized Ads Recommendation Engine

AdFusion is an AI-powered system that delivers personalized product recommendations within visual content streams like photo feeds. It leverages transformer-based embeddings and FAISS (approximate nearest neighbor) search to match user preferences with relevant products in real time.

Internal Code Copilot (RAG System)

This is a full Retrieval-Augmented Generation (RAG) powered internal AI Copilot prototype. It assists engineers working with large codebases, hardware SDKs, APIs, error logs, and design documentation — built for real-world enterprise engineering use cases.

Automated Parking Area Detection

A computer vision system that detects available parking spots in real time using OpenCV. The model processes live or recorded video feeds, applies contour detection and ROI analysis to identify and highlight free and occupied parking spaces automatically.

Smart Attendance System

A dual-mode intelligent attendance system that combines facial recognition and RFID authentication to ensure accurate and secure student or employee tracking. It uses computer vision for identity verification and RFID tags for contactless logging, reducing manual effort and preventing proxy attendance.

Stock News Sentimental Analysis

A web-based stock assistant that provides users with fundamental stock details, daily top gainers/losers, and category-based recommendations (e.g., penny stocks, volume-based picks). It integrates live stock news with sentiment classification (positive, negative, neutral) powered by a machine learning summarizer. Built using HTML/CSS/JS (frontend), PHP & Python (backend), and MySQL for data storage.
